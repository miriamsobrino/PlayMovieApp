# PlayMovieApp
Mobile app developed in android with Kotlin using the TMDb API

## Tech Stack
- Android XML
- Architecture MVVM
- ViewBinding
- Retrofit
- Coroutines
- Dagger Hilt
- Room
- Jetpack Navigation
- Gson 
- Picasso
